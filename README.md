TexmexUploader Workbench Integración CAD para Mantenimiento y Diseño en Texmex Weavers

TexmexUploader es un Workbench desarrollado para FreeCAD que permite al departamento de Mantenimiento, Ingeniería y Diseño:

✔ Subir modelos 3D (FCStd) directamente a servidor S3 ✔ Subir y estandarizar planos SVG de TechDraw ✔ Agregar atributos automáticos a documentos y páginas ✔ Consultar la librería CAD corporativa desde FreeCAD ✔ Descargar, abrir e importar modelos con un clic ✔ Mantener sincronizados los archivos de ingeniería

Todo esto integrado con las herramientas internas de Texmex Weavers.

Guía de Uso — TexmexUploader Workbench

    Subir un modelo 3D (FCStd)

Abrir un archivo o ensamblaje en FreeCAD.

El workbench:

Agrega atributos (autor, revisión, descripción).

Dar clic en Modelos 3D → Subir modelo actual.

Sube el archivo al servidor S3 bajo el área correspondiente.

    Subir un plano SVG (TechDraw)

Crear la página en el banco de dibujo.

Completar atributos del documento (auto).

Dar clic en Planos SVG → Subir página actual.

El Workbench:

Genera metadatos.

Estandariza plantilla SVG.

Envía archivo al bucket de planos.

    Usar la Librería CAD Corporativa

Abrir desde:

Librería CAD → Abrir Librería CAD Texmex Weavers

Funciones disponibles:

✔ Navegar áreas → máquinas → subensambles ✔ Abrir un modelo nuevo ✔ Importar dentro del documento actual ✔ Ver vista previa ✔ Eliminar archivos obsoletos (con permisos) ✔ Botón Sync para actualizar árbol completo
